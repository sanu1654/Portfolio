Multi-Purpose Autonomous Robot

In my Degree Minor project, as a part of a three-member team, I actively contributed to the creation of a remarkable robotic system titled the "Multi-Talented Robot." This innovative project serves as a testament to a wide array of intelligent functionalities, featuring obstacle avoidance, auto path following, and human hand trackingâ€”all seamlessly controllable through an intuitive mobile application.


In the development process, I harnessed the power of an Arduino UNO microcontroller, incorporating advanced sensors and a Bluetooth module for convenient mobile connectivity.This project is the sophisticated integration of cutting-edge sensors, including ultrasonic and infrared sensors, L298N motor drivers, and servo motors.


The standout capabilities of this project include its ability to autonomously navigate and avoid obstacles through the obstacle avoidance mode, making it ideal for automated environments. The auto path following mode enables the robot to adeptly track predefined paths, demonstrating its potential in areas like warehouse logistics. Additionally, the human hand following mode adds an interactive dimension, as the robot can detect and pursue a user's hand.


Its practical applications extend to diverse fields, encompassing robotics research, industrial automation, and human-robot interaction. For my degree's Major Project, I developed an Intelligent Multi-Camera Surveillance System, designed to redefine security standards through the use of advanced technology. The system utilizes the YOLOv8 module in Python for state-of-the-art object detection, segmentation, and tracking, ensuring accurate and efficient surveillance. OpenCV was integrated to enable real-time frame processing and display, while the capabilities of the system were further enhanced using the Ultralytics package. A key feature of this project is its multi-streaming functionality, achieved through the threading module, which allows simultaneous tracking across multiple camera feeds. This intelligent surveillance system represents a significant step forward in security technology by offering comprehensive object tracking and real-time analysis across various camera streams.
